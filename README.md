# Portfolio Website

Static portfolio website for internship and placement use.

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `script.js`: editable content and interactive behavior

## Run locally

Open `index.html` directly in a browser, or run:

```bash
cd "/Users/darshilprajapati/Desktop/Hack & Project/protfolio_web"
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Update before using

Replace placeholder values in `script.js` and `index.html`:

- name and intro text
- email, GitHub, LinkedIn, resume link
- project demo and source links
- skills, achievements, and timeline details
